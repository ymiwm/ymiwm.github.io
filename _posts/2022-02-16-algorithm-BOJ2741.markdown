## 5. Code
```cpp
#include <bits/stdc++.h>

using namespace std;

int N;

void input();
void func();
void output();

int main(void)
{
    input();
    func();
    output();
}

void input()
{
    cin >> N;
}

void func()
{

}

void output()
{
    for(int i = 1; i <= N; i++)
        cout << i << '\n';
}
```

## 6. 정리
나 자신에게만 해당되는 이야기일 수 있지만,  
컨테이너를 사용할때 Index가 0부터 시작하여  
습관적으로 `for`문의 변수를 0부터 사용하는데,  
이런 작은 습관으로 나중에 큰 낭패를 볼 수 있으니 유의하자.
