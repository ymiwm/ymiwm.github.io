`output`부분의 `for`문은 문장이 다소 어지럽지만, 스스로 규칙을 만들어 지키다보니

## 5. Code
```cpp
#include <bits/stdc++.h>

using namespace std;

int N;

void input();
void func();
void output();

int main(void)
{
    input();
    func();
    output();
}

void input()
{
    cin >> N;
}

void func()
{
    
}

void output()
{
    for(int i = 0; i < N; i++)
    {
        for(int j = 0; j < N - i - 1; j++)
            cout << ' ';
        for(int j = N - i - 1; j < N; j++)
            cout << '*';
        cout << '\n';
    }
}
```

## 6. 정리
문제 제한 조건이 단순하여 그대로 구현하면 되는 문제였다.  
단 제한 조건이 까다로워질 경우 int 타입이 아닌 long 타입 혹은 문자열을 고려하자.
